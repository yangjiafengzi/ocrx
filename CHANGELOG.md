# 更新日志

所有重要的更新都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.1.0] - 2026-03-21

### 新增

- **少样本提示功能（Few-shot Prompting）**：支持上传示例图片及其正确识别结果，提高手写笔记等特殊场景的识别准确率
- **示例库管理**：独立的示例库管理界面，支持添加、删除、选择示例
- **示例图片与文本关联**：每个示例包含图片和对应的正确文本
- **多示例支持**：支持选择多个示例（建议1-3个）进行少样本提示
- **增强重试机制**：重试次数从3次增加到5次，提高识别成功率

### 改进

- **参数解包安全性**：使用 `*optional_params` 替代 `len(params)` 判断，提高代码向后兼容性
- **调试日志增强**：添加详细的调试日志，便于问题定位
- **并发处理优化**：修复并发识别时的卡死问题
- **结果保存逻辑**：修复保存逻辑，确保识别结果正确保存

### 修复

- **示例ID截断问题**：修复 Treeview 中示例ID显示截断导致无法正确识别的问题
- **参数传递问题**：修复少样本示例参数在传递过程中丢失的问题
- **空内容判断**：优化 API 返回空内容的处理逻辑
- **语法错误**：修复多处缩进和语法错误

## [2.0.0] - 2026-02-26

### 新增

- **全新架构**：采用处理器模式，代码更清晰、易于维护
- **进度条显示**：实时显示处理进度，包括预处理、识别、合并各阶段
- **提示词预设管理**：支持保存、另存、重命名、删除、重置提示词预设
- **重试机制**：关键操作（OCR、合并）自动重试，提高成功率
- **页面范围选择**：支持指定 PDF 页码范围，如 `1,3,5-10`
- **剪贴板历史**：自动记录复制历史，支持查看和重新复制
- **并发处理优化**：使用线程池，支持多页面并发识别
- **超时控制**：单页面识别超时 180 秒，防止无限等待
- **结构化日志**：使用 JSON 格式日志，便于分析和调试

### 改进

- **界面优化**：使用 Notebook 分页，界面更整洁
- **错误处理**：更详细的错误信息和处理
- **性能优化**：减少内存占用，提高处理速度
- **配置管理**：配置文件自动保存和加载

### 修复

- **复制问题**：修复剪贴板复制失败的问题
- **合并问题**：修复结果合并时格式错误的问题
- **进度计算**：修复多文件处理时的进度计算
- **内存泄漏**：修复图片处理后的内存释放问题

### 变更

- **代码重构**：将功能拆分到独立的处理器中
- **依赖更新**：更新关键依赖库到最新版本
- **API 调整**：统一 API 调用接口

### 移除

- **旧版 GUI**：完全重写 GUI 层
- **冗余代码**：删除重复和无用代码

## 版本说明

### 版本号规则

- **主版本号**：重大更新，可能不兼容
- **次版本号**：新功能，向后兼容
- **修订号**：Bug 修复，向后兼容

### 更新类型

- **Added**：新功能
- **Changed**：功能变更
- **Deprecated**：即将移除的功能
- **Removed**：已移除的功能
- **Fixed**：Bug 修复
- **Security**：安全相关更新
