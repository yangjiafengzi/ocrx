# 更新日志

所有重要的更新都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2026-02-26

### 新增

- **全新架构**：采用处理器模式，代码更清晰、易于维护
- **进度条显示**：实时显示处理进度，包括预处理、识别、合并各阶段
- **提示词预设管理**：支持保存、另存、重命名、删除、重置提示词预设
- **重试机制**：关键操作（OCR、合并）自动重试，提高成功率
- **页面范围选择**：支持指定 PDF 页码范围，如 `1,3,5-10`
- **剪贴板历史**：自动记录复制历史，支持查看和重新复制
- **并发处理优化**：使用线程池，支持多页面并发识别
- **超时控制**：单页面识别超时 180 秒，防止无限等待
- **结构化日志**：使用 JSON 格式日志，便于分析和调试

### 改进

- **界面优化**：使用 Notebook 分页，界面更整洁
- **错误处理**：更详细的错误信息和处理
- **性能优化**：减少内存占用，提高处理速度
- **配置管理**：配置文件自动保存和加载

### 修复

- **复制问题**：修复剪贴板复制失败的问题
- **合并问题**：修复结果合并时格式错误的问题
- **进度计算**：修复多文件处理时的进度计算
- **内存泄漏**：修复图片处理后的内存释放问题

### 变更

- **代码重构**：将功能拆分到独立的处理器中
- **依赖更新**：更新关键依赖库到最新版本
- **API 调整**：统一 API 调用接口

### 移除

- **旧版 GUI**：完全重写 GUI 层
- **冗余代码**：删除重复和无用代码

## [1.x.x] - 早期版本

## 版本说明

### 版本号规则

- **主版本号**：重大更新，可能不兼容
- **次版本号**：新功能，向后兼容
- **修订号**：Bug 修复，向后兼容

### 更新类型

- **Added**：新功能
- **Changed**：功能变更
- **Deprecated**：即将移除的功能
- **Removed**：已移除的功能
- **Fixed**：Bug 修复
- **Security**：安全相关更新
