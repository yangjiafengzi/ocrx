## 快速开始

### 第一步：配置 API

1. 打开程序后，在"主要配置"页面填写：
   - **Base URL**：您的 API 服务地址
   - **API Key**：您的 API 密钥
   - **Model Name**：模型名称（如 gpt-4o）

2. 点击"保存配置"按钮

### 第二步：选择文件

1. 点击"选择文件"按钮
2. 选择要识别的 PDF 或图片文件
3. 支持多文件选择

### 第三步：开始识别

- 点击"识别并保存"：结果保存为 Markdown 文件
- 点击"识别并复制"：结果复制到剪贴板

## 基本操作

### 文件选择

**单文件**：
1. 点击"选择文件"
2. 选择单个 PDF 或图片
3. 点击"打开"

**多文件**：
1. 点击"选择文件"
2. 按住 Ctrl 键选择多个文件
3. 点击"打开"

**清空文件**：
- 点击"清空"按钮

### 输出目录

1. 点击"选择目录"
2. 选择保存结果的文件夹
3. 点击"确定"

### 处理模式

#### 识别并保存

适用场景：
- 批量处理大文档
- 需要保存结果
- 后续需要编辑

操作步骤：
1. 配置 API 和文件
2. 点击"识别并保存"
3. 等待处理完成
4. 结果保存在输出目录

#### 识别并复制

适用场景：
- 快速复制小量内容
- 直接粘贴使用
- 不需要保存文件

操作步骤：
1. 配置 API 和文件
2. 点击"识别并复制"
3. 等待处理完成
4. 结果自动复制到剪贴板

**注意**：识别并复制模式限制 10 页以下文档

### 停止处理

点击"停止"按钮可中断当前处理

## 高级功能

### 提示词预设管理

#### 选择预设

1. 在"OCR 提示词预设"下拉框中选择
2. 自定义提示词编辑区会自动更新

#### 保存更改

修改提示词后，点击"保存更改"按钮

#### 另存为新预设

1. 修改提示词
2. 点击"另存为"按钮
3. 输入新预设名称
4. 点击确定

#### 重命名预设

1. 选择要重命名的预设
2. 点击"重命名"按钮
3. 输入新名称
4. 点击确定

#### 删除预设

1. 选择要删除的预设
2. 点击"删除"按钮
3. 确认删除

**注意**：默认预设（手写笔记、印刷材料）不能删除

#### 重置所有预设

点击"重置所有"按钮，恢复为软件内置的默认预设

### 页面范围（PDF）

**格式说明**：
- 单页：`1,3,5`
- 范围：`1-10`
- 混合：`1,3,5-10,15`

**示例**：
- `1-5`：第 1 到 5 页
- `1,3,5`：第 1、3、5 页
- `1-3,5,7-10`：第 1-3、5、7-10 页

**留空**：处理全部页面

### PDF 缩放比例

**作用**：控制 PDF 转换为图片时的分辨率

**建议值**：
- 文字清晰：2.0-3.0
- 手写笔记：3.0-4.0
- 复杂图形：4.0-5.0

**注意**：值越大，识别精度越高，但处理速度越慢

### 最大并发数

**作用**：控制同时处理的页面数量

**建议值**：
- 普通电脑：5-10
- 高性能电脑：10-20
- 低性能电脑：1-5

**注意**：值越大，处理越快，但占用资源越多

## 常见问题

### Q: 如何获取 API Key？

A: 根据您使用的 AI 服务提供商的指引获取。常见提供商：
- OpenAI：https://platform.openai.com
- Azure OpenAI：https://azure.microsoft.com
- 其他兼容 OpenAI API 的服务

### Q: 支持哪些文件格式？

A: 
- PDF（.pdf）
- 图片（.png, .jpg, .jpeg, .bmp）

### Q: 识别结果保存在哪里？

A: 默认保存在 `Documents/OCRX_Output` 目录，文件名格式：`原文件名_ocr.md`

### Q: 如何查看剪贴板历史？

A: 点击"剪贴板历史"标签页，可以看到最近的复制记录

### Q: 可以自定义提示词吗？

A: 可以，在"自定义提示词"编辑区修改，然后点击"保存更改"或"另存为"

### Q: 识别失败怎么办？

A: 
1. 检查 API 配置是否正确
2. 检查网络连接
3. 查看"运行日志"页面的错误信息
4. 尝试降低 PDF 缩放比例
5. 尝试减少并发数

## 故障排除

### 问题：程序无法启动

**可能原因**：
- Python 版本过低
- 缺少依赖库

**解决方法**：
1. 检查 Python 版本（需要 3.8+）
2. 运行 `pip install -r requirements.txt`

### 问题：API 调用失败

**可能原因**：
- API Key 错误
- Base URL 错误
- 网络问题
- 余额不足

**解决方法**：
1. 检查 API 配置
2. 测试网络连接
3. 检查 API 账户余额

### 问题：复制到剪贴板失败

**可能原因**：
- 剪贴板被其他程序占用
- 内容过大

**解决方法**：
1. 关闭可能占用剪贴板的程序
2. 尝试分批复制
3. 使用"识别并保存"模式

### 问题：PDF 转换失败

**可能原因**：
- PDF 文件损坏
- PDF 加密
- 内存不足

**解决方法**：
1. 检查 PDF 文件是否正常
2. 尝试指定页面范围
3. 降低 PDF 缩放比例

### 问题：进度条卡住

**可能原因**：
- API 响应慢
- 网络超时
- 页面处理超时

**解决方法**：
1. 等待（最长 3 分钟）
2. 点击"停止"后重试
3. 检查网络连接
4. 尝试减少并发数

### 问题：识别结果为空

**可能原因**：
- PDF 是扫描件但提示词不匹配
- 图片质量太差
- 提示词设置不当

**解决方法**：
1. 选择正确的提示词预设
2. 提高 PDF 缩放比例
3. 自定义提示词

## 最佳实践

### 提高识别精度

1. **选择合适的提示词预设**
   - 手写笔记：使用"手写笔记"预设
   - 印刷材料：使用"印刷材料"预设

2. **调整 PDF 缩放比例**
   - 文字小：提高缩放比例
   - 文字大：降低缩放比例

3. **指定页面范围**
   - 只识别需要的页面
   - 避免处理空白页

### 提高处理速度

1. **增加并发数**
   - 根据电脑性能调整
   - 高性能电脑可设置 15-20

2. **降低 PDF 缩放比例**
   - 适当降低可提高速度
   - 但可能影响精度

3. **指定页面范围**
   - 减少处理页数

### 保证稳定性

1. **合理设置并发数**
   - 不要设置过高
   - 避免系统资源耗尽

2. **分批处理大文档**
   - 使用页面范围分批
   - 避免一次性处理过多页面

3. **保存配置**
   - 经常保存配置
   - 避免重复设置
